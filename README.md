# Mon Délicieux — ごちそうさまの記録

> 日記であり、カレンダーであり、手帳であり、一人食べログ。

**[→ アプリを開く](https://jiz41.github.io/mon-delicieux/)**

---

## 機能

- 記録の追加・編集・削除（店名/料理/日付/ジャンル/うまみ度/メモ/写真）
- カレンダー表示（テーマカラーのドット付き）
- 月別ジャンル円グラフ
- うまみ度の店別平均分析
- チラシ出力（PNG ダウンロード）
- ダークモード
- JSON バックアップ / インポート

## データ

すべて `localStorage` に保存。サーバー不要。
